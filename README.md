# CFIP CARBALLEIRA

1. Clonar el repositorio con Visual Studio o similar
2. Abrir el proyecto con Unity 6 (version 6000.2.12f1)
3. Conectar el dispositivo con MyOWO app o abrir el Visualizador > Connect games
4. En el Juego, seleccionar la IP
5. Conectar en la aplicacion o visualizador
6. Sentir la sensaciones implementadas
